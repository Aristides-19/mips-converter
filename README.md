# Formatos de entrada del buffer

Se asume que los datos ingresados siguen los siguientes formatos (no se validan):

- ***Binario***: 32 bits sin espacios (ej. 00000000000000001011001100111100)
  
- ***Empaquetado***: 32 bits con espacios cada 4 bits (ej. 0000 0000 0000 0000 0000 0101 0011 1100)
  
- ***Octal***: Número entero con signo positivo o negativo (ej. +52)
  
- ***Decimal***: Número entero con signo positivo o negativo (ej. -123456)
  
- ***Hexadecimal***: Mayúsculas sin espacios con signo positivo o negativo (ej. -186A0)
  
- ***Fraccionario***: Número con punto decimal y signo negativo opcional (ej. -2.75)
